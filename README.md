# Job Role Prediction System using Machine Learning

## Overview

This project predicts the most suitable job role for a candidate based on their skills, experience, and professional background using Machine Learning techniques.

The project includes data preprocessing, feature engineering, model training, model comparison, and prediction pipelines.
